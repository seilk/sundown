# Defaults and Onboarding Requirements

## Required onboarding fields

User must complete these before normal app use:

- Daily work limit (required)
- Day reset time (required)

## Daily limit input

- Use presets: 6h, 8h, 10h, and Custom.
- No implicit default limit value.
- User must select one value at first run.

## Day reset input

- User-defined local reset time during onboarding.
- Not fixed to midnight.

## Initial default settings

- Idle threshold: 5 minutes.
- Notifications: off by default, user opt-in.
- Over-limit reminder interval (when enabled): 30 minutes.
