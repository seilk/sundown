# Product Requirements Document (MVP)

## Product

- Name: Sundown
- Platform: macOS menubar app
- Users: People who overwork due to lack of time boundaries

## Problem

Many users sit and work for too long with no structured end point, which can lead to poor health and burnout.

## Goal

Make overwork visible in a low-friction way and encourage sustainable work rhythm.

## MVP scope

- Menubar first product.
- Main intervention is visual time state in menubar.
- Daily ritual summary with Work/Break/Idle pie chart.
- Goal system is optional.
- Optional notification system for over-limit reminders.

## Out of scope (MVP)

- Account sync / cloud storage.
- Team features.
- Cross-platform clients.

## Success signals

- User can configure a daily limit and reset time during onboarding.
- User can always see remaining time or over-time in menubar.
- User can review same-day Work/Break/Idle distribution.
- User can enable or disable over-limit notifications.
