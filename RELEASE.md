# Open Source Release Plan

## License

- MIT

## Repository

- GitHub slug: `sundown`

## Release goals

- Publish a useful and minimal menubar tool quickly.
- Keep onboarding strict enough to prevent silent misuse.
- Make contribution process simple and predictable.

## Pre-release checklist

- Confirm docs match actual behavior.
- Add basic screenshots/gifs for first impression.
- Verify app builds on supported macOS versions.
- Verify onboarding enforcement (limit + reset time required).
- Verify over-limit red state and optional notifications.

## Post-release checklist

- Triage first issues and label by type.
- Capture user feedback on pacing UX.
- Prioritize improvements without bloating MVP.
